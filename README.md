# Unbound_hackathon
A lightweight system to create and run multi-step AI workflows where each step is executed by an LLM. Outputs from previous steps are automatically passed as context to the next step.
Create multi-step AI workflows

Choose model and prompt per step

Automatic context passing

Retry on failure

Simple completion criteria

Live execution logs in browser

Uses Unbound API

Tech stack:
Backend: Python + FastAPI

Frontend: HTML (built-in UI)

LLM Provider: Unbound API

HTTP Client: httpx
